# 莫比乌斯反演

## 莫比乌斯函数

### 定义
对于正整数 $n$，莫比乌斯函数 $\mu(n)$ 定义如下：

$$
\mu(n) = 
\begin{cases}
1, & \text{若 } n = 1; \\
(-1)^k, & \text{若 } n \text{ 是 } k \text{ 个不同质数的乘积} \\
& \text{（即 } n \text{ 无平方因子且 } n > 1\text{）}; \\
0, & \text{若 } n \text{ 被某个质数的平方整除} \\
& \text{（即 } n \text{ 有平方因子）}.
\end{cases}
$$
